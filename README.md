## ✨ Features

- Colorful & Creative design with animated gradients
- Custom cursor with smooth follower effect
- Scroll reveal animations
- Skills section with colored pills
- Projects showcase cards
- Contact section with social links
- Fully responsive on all devices
- Dockerized with Nginx for easy deployment

## 🚀 Run Locally with Docker

### 1. Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/portfolio.git
cd portfolio
```

### 2. Build the Docker image
```bash
docker build -t my-portfolio .
```

### 3. Run the container
```bash
docker run -d -p 8080:80 --name portfolio my-portfolio
```

### 4. Open in browser
```
http://localhost:8080
```
